# Claude Code 抖音高密度短视频课程包

## 课程定位

这套课把原始 Claude Code 教程重新编排成一组适合抖音连载的视频脚本。口吻尽量像开发者复盘自己踩过的坑，不做课堂式讲解。

核心不是逐条讲按钮和概念，而是用 6 个常见使用场景串起完整知识体系：Claude Code 为什么不是聊天框、第一次怎么安全启动、怎么写第一条提示词、怎么管理上下文、怎么使用 Subagents / Skills / MCP / Hooks，最后怎么组合成稳定工作流。

项目实践只在需要解释知识点时出现，作为辅助画面和例子，不再作为每集主角。

## 目标观众

- 会写一点代码，但还没系统用过 Claude Code 的开发者。
- 想把 AI 从“问答工具”升级成“项目协作者”的独立开发者。
- 正在做 AI 产品、内容工具、SaaS 工具，希望学习 AI 辅助开发流程的产品/技术创作者。

## 抖音调性

- 每集 2-4 分钟，信息密度高，但用故事推进，不做条目朗读。
- 开场尽量从事故、悬念或反常识切进去。
- 口播像讲案子：先给异常，再调查原因，再回收知识点。
- 技术点用“现场证据”解释：权限、上下文、工具调用、计划、验证、自动化。
- 项目画面只在必要时出现，用于展示真实文件、计划、测试或工作流。

## 系列结构

1. 我让 Claude Code 随便看看，它差点开始上班
2. 我说加个暗黑模式，它差点把项目送走
3. 它昨天刚会，今天又不会了
4. 我派了三个子代理，结果像拉了三个群聊
5. 我复制同一段提示词复制到像个快捷键
6. 别找神提示词了，先把流程搭起来
